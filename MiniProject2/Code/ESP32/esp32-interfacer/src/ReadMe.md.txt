Check if the esp32 is showing in the COM port not in other devices in Windows Device Manager.

If it is showing under "other devices" install "CP210x Universal Windows Driver".

The link -> https://www.silabs.com/developer-tools/usb-to-uart-bridge-vcp-drivers?tab=downloads

Unzip the file in the location you have installed Arduino.
